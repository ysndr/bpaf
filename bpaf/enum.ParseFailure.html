pub enum ParseFailure {
    Stdout(String),
    Stderr(String),
}
Expand description

Unsuccessful command line parsing outcome, use it for unit tests

Useful for unit testing for user parsers, consume it with ParseFailure::unwrap_stdout and ParseFailure::unwrap_stdout

Variants

Stdout(String)

Print this to stdout and exit with success code

Stderr(String)

Print this to stderr and exit with failure code
